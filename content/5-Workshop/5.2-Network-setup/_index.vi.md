Để xây dựng hạ tầng mạng tối ưu, bảo mật và có khả năng tự động mở rộng (Auto Scaling) cho hệ thống Web bán lẻ thời trang kết hợp Pipeline Machine Learning, chúng ta sẽ thực hiện cấu hình các thành phần mạng trên AWS bao gồm: CloudFront Distribution, Load Balancers (ALB), Target Groups, các máy chủ Web Server chạy trên EC2 và Auto Scaling Group (ASG).
