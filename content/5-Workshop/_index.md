# Building a Fashion Retail Web App & ML Pipeline for Sales Forecasting

#### Overview

In this workshop, we will build a complete fashion retail web application architecture on **AWS Cloud**, integrated with an automated machine learning pipeline that handles data extraction, feature engineering, and sales forecasting.

The system is designed to optimize inventory management by forecasting customer shopping demand based on historical transaction data. We will gain hands-on experience deploying network infrastructure, load balancers, databases, and heavy-duty big data services like **AWS Glue**, **Amazon EC2**, **Amazon S3**, and **AWS Lambda**.
